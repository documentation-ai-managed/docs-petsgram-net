<Update label="2024-11-15" description="v2.0.0" tags={["feature", "improvement"]}>

## New Features

- **Enhanced Hashtag Feed**: Discover trending pet hashtags with real-time filtering. Use `#MyPetAdventure` to feature your posts on the main page.
- **Store Checkout API**: Integrate seamless purchases with new webhooks for order notifications.

## Improvements

- Faster image uploads by `>50%` for smoother sharing.
- Redesigned store categories for better pet product discovery.

## Bug Fixes

- Fixed hashtag search returning duplicate posts.
- Resolved store cart persistence issues on mobile.

Example webhook payload for new orders:

<CodeGroup tabs="JavaScript,Python">
```javascript
// Handle incoming webhook
app.post('/webhook/orders', (req, res) => {
  const { orderId, petProduct, total } = req.body;
  console.log(`New order ${orderId}: ${petProduct} for $${total}`);
  res.status(200).send('OK');
});
```
```python
# Handle incoming webhook
from flask import Flask, request

app = Flask(__name__)

@app.route('/webhook/orders', methods=['POST'])
def orders_webhook():
    data = request.json
    order_id = data['orderId']
    product = data['petProduct']
    total = data['total']
    print(f"New order {order_id}: {product} for ${total}")
    return 'OK', 200
```
</CodeGroup>
